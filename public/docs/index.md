# URDOC 平台文档

欢迎使用 URDOC 文档中心。URDOC 是一款基于大语言模型驱动的 AI 平台，面向医学教育领域，围绕 **CSTAR** 教学框架，提供从病例生成、叙事写作、考核出题到模拟问诊的全流程工具。

## 关于 URDOC

URDOC 致力于为医学教育领域提供 AI 驱动的虚拟病例研究平台。通过 CSTAR 教学框架，帮助医学生和教师实现从理论到实践的完整教学闭环。
