# 项目背景

## 设计理念

URDOC 旨在通过 AI 技术赋能医学教育，将大语言模型的能力与 CSTAR 教学框架深度融合，打造一个开放、灵活、智能的虚拟病例研究平台。

### 核心理念

- **AI 驱动** — 基于大语言模型，实现病例生成、叙事创作、模拟问诊等核心功能
- **教学融合** — 紧密围绕医学教育实际需求设计，与 CSTAR 教学框架完美契合
- **多模态支持** — 覆盖文本、图像、视频、语音多种表达形式
- **开放架构** — 支持多种 AI 模型接入，灵活适应不同场景

## 技术栈

| 层面 | 技术 |
|------|------|
| 前端框架 | Nuxt 4 + Vue 3 |
| UI 组件库 | Nuxt UI v4 (Tailwind CSS v4) |
| 状态管理 | Pinia |
| AI SDK | ai + @ai-sdk/openai-compatible |
| 内容渲染 | @comark/nuxt (Comark) |
| 部署平台 | EdgeOne 无服务器函数 |

## 发展路线

URDOC 将持续演进，未来的发展方向包括：

- 更多类型的病例和科室覆盖
- 增强的多模态能力
- 更完善的评估体系
- 协作与分享功能
- 移动端适配
