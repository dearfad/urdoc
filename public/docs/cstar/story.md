# 编写故事

基于病例创作医疗叙事故事，体现叙事医学理念，培养同理心与人文关怀。

## 功能说明

叙事医学强调通过理解患者的疾病经历来提升临床能力。本模块基于已生成的病例，AI 自动创作以患者为主线的医疗叙事故事。

## 使用步骤

### 1. 准备病例

需要一个已生成的病例作为故事基础。你可以：

- **使用当前病例** — 如果已在该会话中生成病例，可直接使用
- **导入已有病例** — 从历史记录中选择

### 2. 配置参数

| 参数 | 说明 |
|------|------|
| **叙事视角** | 患者视角、医生视角或第三人称视角 |
| **故事风格** | 专业纪实、文学叙事、医患对话等 |
| **篇幅** | 短篇、中篇或长篇 |

### 3. 生成故事

点击生成按钮，AI 将基于病例信息创作医疗叙事故事。故事通常包含：

- 患者的背景和症状出现过程
- 就医经历和诊疗过程
- 心理变化和情感体验
- 疾病对生活和家庭的影响
- 治疗结果和康复展望

### 4. 查看与操作

生成完成后可以查看、复制和重新生成故事内容。

## 教学应用

- 用于医学人文课程讨论
- 帮助医学生理解患者心理
- 培养共情能力和医患沟通技巧
- 作为叙事医学教学案例
