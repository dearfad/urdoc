# 评估能力

多维度评估临床能力，持续追踪学习进度与成长曲线。

## 功能说明

评估能力模块提供多维度的临床能力评估体系，覆盖诊断准确性、治疗方案合理性、沟通能力等多个方面，帮助学习者和教师了解学习进展。

## 评估维度

| 维度 | 说明 |
|------|------|
| **诊断能力** | 病史采集完整性、诊断推理逻辑性、鉴别诊断全面性 |
| **治疗能力** | 治疗方案合理性、用药规范性、预后判断准确性 |
| **沟通能力** | 问诊技巧、医患沟通、信息采集效率 |
| **知识掌握** | 理论知识应用、医学概念理解、循证医学能力 |

## 使用步骤

### 1. 选择评估对象

选择要评估的互动实践记录或考核结果。

### 2. 进行评估

AI 将根据记录内容进行分析评估，生成评估报告，包含：

- 各维度得分（百分制）
- 文字评语
- 改进建议
- 与历史表现对比的趋势分析

### 3. 查看报告

评估完成后可以查看详细的评估报告，了解优势和待改进的领域。

## 学习追踪

评估结果会自动保存，形成个人学习成长曲线，支持：

- 查看历次评估结果的趋势变化
- 发现薄弱环节并针对性改进
- 跟踪学习进步

## 教学应用

- **自我评估** — 了解自身临床能力水平
- **教师评估** — 辅助教师进行学生能力评估
- **教学改进** — 根据评估结果调整教学重点
