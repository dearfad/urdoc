# 生成病例

AI 自动生成符合执业医师考试标准的完整结构化病例，是 CSTAR 教学闭环的起点。

## 功能说明

生成病例模块利用大语言模型的医学知识，生成真实感强的临床病例。每个病例包含完整的结构化信息，适用于临床教学和自主学习。

## 使用步骤

### 1. 配置参数

在右侧设置面板中配置以下参数：

| 参数 | 说明 |
|------|------|
| **患者性别** | 选择病例中患者的性别 |
| **患者年龄** | 设置患者年龄范围 |
| **就诊方式** | 门诊、急诊或住院 |
| **科室** | 选择目标科室（如内科、外科、妇产科、儿科等） |
| **主要诊断** | 设定核心疾病诊断 |
| **并发症** | 可选并发症，增加病例复杂度 |
| **其他要求** | 自定义附加要求 |

#### 内容来源

生成病例的材料来源于以下两种方式：

##### 教科书内容

按照教科书目录层次结构，可以手动或者随机选择 **教科书 → 篇目 → 章节 → 节次 → 子节 → 主题**。当前可用的教科书如下：

|  教科书  | 出版时间 | 版次 | 大小 |
| :------: | :------: | :--: | :--: |
|  外科学  | 2024.07  |  10  | 27KB |
|  内科学  | 2024.07  |  10  | 17KB |
| 妇产科学 | 2024.06  |  10  | 8KB  |
|  儿科学  | 2024.06  |  10  | 13KB |
| 神经病学 | 2024.07  |  9   | 13KB |

如需自定义教科书，请参考[自定义教科书](/docs/development/custom-textbook)。

##### 自定义内容

- 在输入框内输入自定义内容，并按回车确认，可添加多个自定义内容
- 已经预置多个内容，可在选择框选择单项或多项内容后，点击添加

### 2. 生成病例

点击生成按钮，AI 将根据配置参数生成完整的结构化病例，包括：

- 基本信息（性别、年龄、就诊方式）
- 主诉
- 现病史
- 既往史
- 体格检查
- 辅助检查
- 诊断与鉴别诊断
- 治疗方案

### 3. 查看与操作

生成完成后可以：

- **查看** — 浏览完整的病例内容
- **复制** — 一键复制病例内容到剪贴板
- **重新生成** — 调整参数后重新生成

## 最佳实践

- **由简到繁**：初学者建议从单一诊断开始，逐步尝试复杂病例
- **贴近教学**：根据实际教学需求设定科室和诊断类型
- **用于考核**：生成的病例可直接用于后续的理论考核和互动实践环节
