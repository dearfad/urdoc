# 考核理论

根据病例自动生成符合考试要求的选择题，高效检验学习效果。

## 功能说明

考核理论模块基于病例内容和相关医学知识，自动生成多种类型的选择题，支持自定义难度和考核重点，适用于形成性评价和总结性评价。

## 使用步骤

### 1. 准备病例

需要一个已生成的病例作为考核基础。

### 2. 配置参数

| 参数 | 说明 |
|------|------|
| **题型** | A1 型（单句型最佳选择题）、A2 型（病例摘要型最佳选择题）等 |
| **题目数量** | 每次生成的题目数量 |
| **难度** | 简单、中等或困难 |
| **考核重点** | 可选择侧重基础知识、诊断推理或治疗方案等 |

### 3. 生成试题

AI 将根据病例信息和配置参数生成选择题，每道题包括：

- 题目内容（题干）
- 选项（通常为 4-5 个）
- 正确答案标识
- 答案解析

### 4. 查看与操作

生成完成后可以查看、复制和重新生成试题。

## 教学应用

- **课堂测验** — 根据教学内容快速生成随堂测验
- **模拟考试** — 模拟执业医师考试题型和难度
- **自测练习** — 学生自主测试知识掌握程度
