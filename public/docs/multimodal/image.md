# 图像创作

AI 生成医学相关图像，为教学和研究提供视觉素材。

## 功能说明

图像创作模块利用 AI 图像生成能力，根据文字描述生成医学相关图像，适用于教学插图、病例配图、解剖示意等场景。

## 使用步骤

### 1. 配置参数

| 参数 | 说明 |
|------|------|
| **提示词** | 描述需要生成的图像内容（支持中文和英文） |
| **风格** | 选择图像风格（真实照片风格、医学插画风格、示意图等） |
| **尺寸** | 图像分辨率 |

### 2. 生成图像

点击生成按钮，AI 将根据提示词生成图像。

### 3. 查看与操作

生成完成后可以：

- **查看** — 查看生成的图像
- **下载** — 保存图像到本地
- **重新生成** — 调整提示词后重新生成

## 提示词技巧

- **具体清晰**：描述主题、动作、环境、构图等要素
- **医学风格**：添加 "medical diagram"、"anatomical illustration" 等关键词
- **参考案例**：
  - ✓ "心脏解剖示意图，显示四个腔室和主要血管，医学插画风格"
  - ✓ "肺部 CT 影像，显示右上叶结节，带有标注"
