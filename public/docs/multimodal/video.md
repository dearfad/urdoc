# 影像渲染

姿态与动作相关影像生成，支持医学操作演示和患者体征动画。

## 功能说明

影像渲染模块利用 AI 影像生成能力，根据描述生成动画或影像内容，适用于展示医学操作流程、患者体征表现等动态场景。

## 使用步骤

### 1. 配置参数

| 参数 | 说明 |
|------|------|
| **描述文本** | 描述需要生成的视频内容 |
| **时长** | 视频长度 |
| **风格** | 动画风格或写实风格 |

### 2. 生成视频

点击生成按钮，AI 将根据配置参数生成视频。

### 3. 查看与操作

生成完成后可以预览和下载视频。

## 应用场景

- **医学操作演示** — 展示标准操作流程（如心肺复苏、穿刺等）
- **患者体征动画** — 模拟震颤、步态异常等体征表现
- **解剖动态展示** — 展示器官结构和生理过程
