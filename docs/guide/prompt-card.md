# 提示词管理

## 类型

| 提示词类型 | 说明                                                   |
| ---------- | ------------------------------------------------------ |
| case       | 用于生成虚拟病例                                       |
| story      | 根据虚拟病例生成各种类型的故事情节                     |
| test       | 根据虚拟病例生成各类测试题目                           |
| act        | 提示大模型模拟患者进行交互                             |
| rate       | 根据测试题模拟教师或患者评估能力                       |
| face       | 根据虚拟病例生成患者外表                               |
| pose       | 根据虚拟病例及外表，生成可能的肢体语言体现疾病特征     |
| review     | 对虚拟病例进行多模型审查，审核虚拟病例的客观性         |
| verify     | 对生成的测试题目进行多模型审查，评估题目和答案的正确性 |

## 使用方法

- **默认**：使用预制提示词进行操作
- **自定义提示词**：
  - 所有用户可新建提示词进行相关操作，
  - 仅注册用户能够保存提示词，默认提示词公开，可选择设置取消公开。
- **刷新**：从数据库中获取所有公开和用户的提示词
